# MCP Configs

Ignav's hosted MCP server is:

```text
https://ignav.com/mcp
```

## Claude Code

```bash
claude mcp add --transport http ignav https://ignav.com/mcp
```

## Claude

In Claude web or desktop settings, add a custom connector with this URL:

```text
https://ignav.com/mcp
```

## Cursor

Copy `mcp.json` into your Cursor MCP config location.
